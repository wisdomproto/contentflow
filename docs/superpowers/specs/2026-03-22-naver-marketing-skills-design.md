# 네이버 마케팅 스킬 설계 문서

**Date**: 2026-03-22
**Status**: Approved
**Target User**: 1인 사업자 / 스몰비즈
**Approach**: 개별 .md 스킬 파일 (기존 Google/Meta 스킬과 동일 포맷)
**Install Location**: `~/.claude/commands/`

---

## 개요

네이버 광고 생태계 전반을 커버하는 12개의 Claude Code 스킬을 만든다.
1인 사업자가 쉽게 사용할 수 있도록 쉬운 용어와 단계별 가이드를 제공하며,
데이터를 넣으면 분석해주고, 데이터 없이도 전략 가이드를 제공하는 하이브리드 방식이다.

## 스킬 목록

### 카테고리 1: 키워드 리서치

#### N01 — 네이버 키워드 리서치 (`naver-keyword-research`)
- **기능**: 네이버 검색광고 API를 통해 키워드별 월간 검색량(PC/모바일), 경쟁률, 파워링크 평균 클릭비용(CPC)을 조회하고 분석
- **API 연동**:
  - 엔드포인트: `https://api.searchad.naver.com/keywordstool`
  - 인증: CUSTOMER_ID, API_KEY, SECRET_KEY (사용자 입력)
  - Python 스크립트로 API 호출 → 결과 테이블 출력
  - API 키 없는 경우: 수동 데이터 입력 모드로 폴백
- **출력**: 키워드별 검색량/경쟁률/CPC 테이블, 업종/시즌별 추천 키워드
- **프로세스**:
  1. API 키 확인 (없으면 발급 가이드 제공)
  2. 업종/시드 키워드 입력받기
  3. API 호출로 데이터 조회
  4. 검색량 기준 정렬, 경쟁률 대비 가성비 키워드 추천
  5. 시즌성 키워드 알림 (월별 검색량 추이)
- **기준값**:
  | 지표 | 좋음 | 주의 | 위험 |
  |------|------|------|------|
  | 월간 검색량 | 1,000+ | 100~999 | <100 |
  | 경쟁률 | 낮음 | 중간 | 높음 |
  | CPC | 업종 평균 이하 | 평균 | 평균 1.5배 이상 |

#### N02 — 네이버 키워드 그룹 플래너 (`naver-keyword-group-planner`)
- **기능**: 핵심 키워드에서 연관 키워드를 확장하고 그룹핑, 예산별 우선순위 추천
- **입력**: 시드 키워드 + 월 예산
- **프로세스**:
  1. N01의 키워드 데이터 활용 (또는 수동 입력)
  2. 키워드를 의도별 그룹핑 (정보탐색 / 비교검토 / 구매의도)
  3. 그룹별 예상 비용 산출
  4. 예산 내 최적 키워드 조합 추천
- **출력**: 그룹별 키워드 리스트, 예상 CPC/클릭수, 예산 배분 시뮬레이션

### 카테고리 2: 유료 광고

#### N03 — 파워링크 세팅 가이드 (`naver-powerlink-setup`)
- **기능**: 파워링크 캠페인을 처음 세팅하는 1인 사업자를 위한 단계별 가이드
- **프로세스**:
  1. 업종/상품 정보 입력받기
  2. 캠페인 구조 설계 (캠페인 → 광고그룹 → 키워드/소재)
  3. 키워드 매치 타입 추천 (정확/구문/확장)
  4. 입찰가 전략 (수동 vs 자동, 업종별 권장 시작가)
  5. 광고 소재 작성 (제목 15자, 설명 45자 규격)
  6. 확장소재 세팅 (부가정보, 추가제목, 홍보문구)
  7. 일 예산 설정 가이드
- **출력**: 세팅 체크리스트, 소재 예시 3~5개, 입찰가 추천

#### N04 — 파워링크 성과 분석 (`naver-powerlink-analysis`)
- **기능**: 네이버 광고관리자에서 내보낸 데이터를 분석하여 최적화 포인트 발굴
- **입력**: CSV/텍스트 데이터 (캠페인/광고그룹/키워드 리포트)
- **프로세스**:
  1. 데이터 파싱 및 구조 파악
  2. 핵심 지표 분석 (노출/클릭/CTR/CPC/전환/전환비용)
  3. 낭비 키워드 발굴 (클릭 多 + 전환 0)
  4. 소재별 CTR 비교
  5. 시간대/요일별 성과 분석
  6. 우선순위별 최적화 액션 플랜 제공
- **기준값**:
  | 지표 | 양호 | 개선필요 | 긴급 |
  |------|------|----------|------|
  | CTR | >2% | 1~2% | <1% |
  | 전환율 | >5% | 2~5% | <2% |
  | CPC | 업종 평균 이하 | 평균 1~1.5배 | 평균 1.5배 초과 |
  | 품질지수 | 7~10 | 4~6 | 1~3 |
- **출력**: 성과 요약 대시보드, 낭비 키워드 리스트, 우선순위별 액션 플랜

#### N05 — 배너광고(GFA) 전략 (`naver-gfa-strategy`)
- **기능**: 네이버 성과형 디스플레이 광고(GFA) 세팅 및 운영 전략
- **프로세스**:
  1. 캠페인 목표 선택 (트래픽/전환/앱설치/동영상)
  2. 타겟팅 설계 (인구통계/관심사/리타겟팅/맞춤타겟)
  3. 소재 규격 안내 (이미지 사이즈별, 네이티브, 동영상)
  4. 입찰 전략 (CPC/CPM/oCPC)
  5. 예산 배분 전략
- **주요 소재 규격**:
  | 유형 | 사이즈 | 권장 |
  |------|--------|------|
  | 배너 이미지 | 1200x628, 750x500 등 | 1200x628 필수 |
  | 네이티브 이미지 | 정사각 1:1 | 권장 |
  | 동영상 | 16:9, 최대 60초 | 15~30초 권장 |
- **출력**: 타겟팅 설계서, 소재 체크리스트, 예산 배분표

#### N06 — 쇼핑검색 광고 최적화 (`naver-shopping-ads`)
- **기능**: 스마트스토어 기반 쇼핑검색 광고 최적화
- **프로세스**:
  1. 상품 피드 품질 체크 (상품명, 카테고리, 이미지, 속성)
  2. 입찰가 관리 (상품별/그룹별)
  3. ROAS 분석 및 목표 설정
  4. 제외 키워드 관리
  5. 시즌/프로모션 전략
- **상품명 최적화 규칙**:
  - 브랜드 + 핵심키워드 + 상품특성 + 옵션 (50자 이내)
  - 특수문자, 과도한 키워드 반복 금지
- **출력**: 상품 피드 개선 리스트, 입찰가 조정 추천, ROAS 시뮬레이션

#### N07 — 브랜드검색 광고 기획 (`naver-brand-search`)
- **기능**: 브랜드검색 광고 소재 구성 및 전략 수립
- **프로세스**:
  1. 브랜드명/주요 키워드 분석
  2. 메인 소재 구성 (썸네일, 제목, 설명, 서브링크)
  3. 서브링크 전략 (이벤트/상품/리뷰/매장안내 등 최대 4개)
  4. 경쟁사 방어 전략 (자사명 + 경쟁사명 검색 대응)
  5. 시즌별 소재 교체 일정
- **출력**: 브랜드검색 소재 기획서, 서브링크 구성안, 연간 운영 캘린더

### 카테고리 3: 바이럴/콘텐츠 마케팅

#### N08 — 브랜드 블로그 콘텐츠 전략 (`naver-brand-blog`)
- **기능**: 자체 네이버 블로그 운영 전략 및 SEO 최적화
- **프로세스**:
  1. 업종/타겟 고객 분석
  2. 키워드별 콘텐츠 유형 매핑 (정보성/후기/비교/가이드)
  3. C-Rank(창작 지수) 최적화 전략
  4. D.I.A(문서 품질 알고리즘) 맞춤 글 구조
  5. 월간 포스팅 캘린더 생성
  6. 포스팅 템플릿 제공
- **D.I.A 최적화 체크리스트**:
  - 본문 1,500자 이상
  - 이미지 3장 이상 (직접 촬영 권장)
  - 동영상 1개 이상 포함 시 가산
  - 소제목(H2/H3) 활용
  - 내부 링크 2개 이상
- **출력**: 월간 콘텐츠 캘린더, 포스팅 템플릿 3~5개, SEO 체크리스트

#### N09 — 블로그 기자단/체험단 관리 (`naver-blog-campaign`)
- **기능**: 블로거 체험단/기자단 캠페인 기획부터 성과 측정까지
- **프로세스**:
  1. 캠페인 목표/예산 설정
  2. 블로거 모집 공고 문구 작성
  3. 선정 기준 가이드 (일 방문자 수, 블로그 지수, 카테고리 적합성)
  4. 블로거 가이드라인 작성 (필수 키워드, 사진 가이드, 금지사항)
  5. 원고 검수 체크리스트
  6. 성과 측정 (노출, 검색 순위, 유입수)
- **주의사항**: 공정위 표시광고법 준수 — "소정의 원고료/제품을 제공받아 작성" 문구 필수
- **출력**: 모집 공고 템플릿, 가이드라인 문서, 검수 체크리스트, 성과 리포트 양식

#### N10 — 네이버 지식인 마케팅 (`naver-kin-marketing`)
- **기능**: 지식인을 활용한 자연스러운 마케팅 전략
- **프로세스**:
  1. 타겟 키워드별 지식인 질문 트렌드 분석
  2. 자연스러운 답변 작성 가이드 (정보 제공 → 자연스러운 추천)
  3. 질문 발굴 (업종별 자주 묻는 질문 리스트)
  4. 답변 퀄리티 체크리스트
- **어뷰징 방지 주의사항**:
  - 동일 IP 다중 계정 금지
  - 자문자답 패턴 회피
  - 과도한 링크 삽입 금지
  - 광고성 답변 패널티 주의
- **출력**: 타겟 질문 리스트, 답변 템플릿 5~10개, 주의사항 체크리스트

#### N11 — 네이버 카페 마케팅 (`naver-cafe-marketing`)
- **기능**: 네이버 카페를 활용한 마케팅 전략
- **프로세스**:
  1. 업종별 타겟 카페 선정 기준 (회원수, 활성도, 관련성)
  2. 자연스러운 게시글 작성 가이드 (정보/후기/질문 형태)
  3. 자체 카페 운영 전략 (회원 모집, 콘텐츠 운영, 등급제)
  4. 카페 내 이벤트 기획
- **출력**: 타겟 카페 리스트 기준, 게시글 템플릿, 자체 카페 운영 플랜

#### N12 — 연관검색어/자동완성 분석 (`naver-related-keywords`)
- **기능**: 네이버 연관검색어 및 자동완성 키워드 현황 분석과 콘텐츠 전략 연계
- **프로세스**:
  1. 브랜드명/주요 키워드의 연관검색어 현황 파악
  2. 긍정/부정 연관검색어 분류
  3. 경쟁사 연관검색어 비교
  4. 연관검색어 트렌드 분석 (월별 변화)
  5. 콘텐츠 전략 연계 (부정 키워드 대응, 긍정 키워드 강화)
- **출력**: 연관검색어 현황표, 긍정/부정 분류, 대응 콘텐츠 전략

- 본문은 한국어 기본, 섹션 제목은 영문+한국어 병기

### 언어/톤
- 한국어 기본
- 1인 사업자 눈높이에 맞춘 쉬운 설명
- 전문 용어 사용 시 괄호 안에 쉬운 설명 병기
- 예: "CPC(클릭당 비용)"

### API 연동 (N01)
- **엔드포인트**: `https://api.searchad.naver.com/keywordstool`
- **인증**: HMAC-SHA256 서명 필요
  - 필수 헤더: `X-Timestamp`, `X-API-KEY`, `X-Customer`, `X-Signature`
  - Signature = HMAC-SHA256(SECRET_KEY, `{timestamp}.{method}.{uri}`)
- **Python 의존성**: `requests` (표준 라이브러리 `hashlib`, `hmac`, `time` 사용)
- **에러 처리**:
  - API 키 미입력 → 발급 가이드 제공 후 수동 입력 모드로 폴백
  - 401/403 → "API 키가 올바르지 않습니다. 네이버 검색광고 > API 관리에서 확인하세요"
  - 429 Rate Limit → "일일 호출 한도 초과. 내일 다시 시도하거나 수동으로 데이터를 입력하세요"
  - 부분 데이터 (일부 키워드 CPC 없음) → 해당 키워드 표시하되 "-" 처리, 나머지 정상 출력
- **API 키 발급 가이드**: 네이버 검색광고 센터 > 도구 > API 사용 관리 경로 안내 포함
- **환경변수 지원**: `NAVER_AD_CUSTOMER_ID`, `NAVER_AD_API_KEY`, `NAVER_AD_SECRET_KEY`

### 데이터 분석 (N04, N06)
- CSV/텍스트 붙여넣기 방식
- 네이버 광고관리자 내보내기 형식 파싱
- 데이터 없이도 일반적인 가이드 제공 가능

### N04 기준값 보충
- CTR/전환율 기준은 **검색광고 기준**이며, 업종에 따라 차이가 큼
- 스킬 실행 시 업종을 먼저 확인하고 업종별 벤치마크 제시
- 업종별 기본 CPC 참고값 (스킬 내 포함):
  | 업종 | 평균 CPC | CTR 벤치마크 |
  |------|----------|-------------|
  | 음식/카페 | 200~500원 | 2~4% |
  | 병원/의료 | 1,000~5,000원 | 1~2% |
  | 교육/학원 | 500~2,000원 | 1.5~3% |
  | 쇼핑몰/패션 | 300~1,000원 | 2~3.5% |
  | 법률/세무 | 2,000~10,000원 | 0.8~1.5% |
  | IT/소프트웨어 | 500~3,000원 | 1~2.5% |

### N10~N12 상세 기준값

**N10 지식인 — 품질 기준**:
| 지표 | 우수 | 보통 | 미달 |
|------|------|------|------|
| 답변 길이 | 300자+ | 100~300자 | <100자 |

| 채택률 목표 | >30% | 10~30% | <10% |

**N11 카페 — 타겟 선정 기준**:
| 지표 | 우수 | 보통 | 비추천 |
|------|------|------|--------|
| 회원 수 | 10만+ | 1만~10만 | <1만 |
| 일 게시글 | 50+ | 10~50 | <10 |
| 업종 연관성 | 직접 관련 | 간접 관련 | 무관 |

**N12 연관검색어 — 데이터 수집 방식**:
- 네이버 검색창에 키워드 입력 후 자동완성/연관검색어를 수동 수집하여 입력
- 사용자가 텍스트로 붙여넣기 → Claude가 분류/분석
- 주기적 모니터링 권장 (주 1회)
